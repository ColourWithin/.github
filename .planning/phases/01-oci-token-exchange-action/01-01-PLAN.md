**Function: `exchange_with_retry(get_jwt_fn, domain_url: str, client_id: str, client_secret: str)`**
Implement the retry loop per D-06 and RESEARCH.md Pattern 1. The loop wraps `TokenExchangeSigner.__init__` (NOT a separate method call — the constructor is eager and does the POST). Use exactly this pattern:

```python
last_exc = None
for attempt, delay in enumerate(DELAYS):
    try:
        return oci.auth.signers.TokenExchangeSigner(
            get_jwt_fn, domain_url, client_id, client_secret
        )
    except oci_requests.exceptions.HTTPError as e:
        status = e.response.status_code if e.response is not None else 0
        if status < 500:
            # 4xx: surface body verbatim and abort — never retry (TOKEX-10)
            body = e.response.text if e.response is not None else str(e)
            print(f"::error::OCI token endpoint error {status}: {body}", flush=True)
            sys.exit(1)
        last_exc = e
    except (oci_requests.exceptions.ConnectionError, OSError) as e:
        last_exc = e
    if attempt < len(DELAYS) - 1:
        time.sleep(delay)
raise RuntimeError(f"OCI token exchange failed after {MAX_ATTEMPTS} attempts") from last_exc
```
