# GitHub Actions

Consumer documentation for ColourWithin first-party GitHub actions.

## Available actions

- [OCI Token Exchange](./oci-token-exchange.md) - Exchange a GitHub Actions OIDC token for an OCI UPST and prepare OCI CLI security-token authentication.
- [Run OCI CLI Command](./run-oci-cli-command.md) - Run `oci` commands with install/version checks, argv-safe execution, output capture, and exit-code handling.
